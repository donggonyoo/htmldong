선택자란?

특정 html 요소를 선택하고자 할 때 사용하는기능
선택자로 해당요소를 선택하면 원하는 "스타일"과"기능" 적용가능

1. 모든(전체) 선택자 : *

    * {
        스타일속성:값;
        스타일속성:값;
    }

2. 태그 선택자 : 태그명

현재 이 문서상에 해당 태그들을 다 선택하고자 할 때 사용

    태그명{
        스타일속성:값;
    }

3. 아이디 선택자 : #아이디명

현재 이 문서상에 특정 html요소 "딱 한개만"을 선택하고자 할때 사용
단, 해당 요소에 id속성을 이용해서 고유한 아이디값 부여해야됨 

    #아이디명{
        스타일속성:값;
    }

해당 이 문서내에 내가 원하는 요소 "여러개"를 함께 선택하고자 할때 사용
